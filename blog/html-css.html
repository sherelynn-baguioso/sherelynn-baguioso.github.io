Classes vs IDs

We use CSS to style our elements and the way to target specific elements is to use selectors. Classes and IDs are two of those selectors. Let us distinguish between the two.

CLASS SELECTORS

Remember that when selecting an element in CSS by its class, a (.) must be added at the beginning of the class name. For the class city, the selector is .city for the ruleset.

ID

If you want an element to be styled uniquely, you can give it an id attribute. However, in contrast to class attribute in which you can use it multiple times, an id can only be applied to a single element, can only have a single value, and can only be used once per page.

To select an id of an element in CSS, we use the id name prepended by a number sign (#). The id name above is large-title therefore the CSS selector is #large-title.

ID selectors have higher specificity than class attributes. Therefore, styles under id selectors will be used instead of class selectors.
